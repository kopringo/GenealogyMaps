Virtualenv setup
================

```bash
python3 -m venv .virtualenv
source .virtualenv/bin/activate
```

Installation
============

```bash
pip3.5 install -r requirements.txt
python3.5 manage.py migrate
ython3.5 manage.py createsuperuser
```
